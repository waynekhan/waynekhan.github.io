# Fun with Vagrant

Assuming [Vagrant](https://www.vagrantup.com/downloads.html) is installed and available on your system:

```bash
$ vagrant -version
Vagrant 2.2.7
$ cd /path/to/your/project/directory/
$ vagrant init hashicorp/bionic64
$ vagrant up
```
