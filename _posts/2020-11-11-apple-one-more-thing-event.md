## Apple One More Thing Event

YouTube video page: [https://www.youtube.com/watch?v=5AwdkGKmZ0I](https://www.youtube.com/watch?v=5AwdkGKmZ0I).

M1 is an SoC - system on a chip with a unified memory architecture:

![M1 overview]({{ site.url }}/assets/m1-overview.png)

8-core CPU: 4 performance/efficiency cores each:

![CPU performance vs. power]({{ site.url }}/assets/cpu-performance-vs-power.png)

This performance gulf seems to be the case for graphics as well, with an (up to) 8-core GPU.

![GPU performance vs. power]({{ site.url }}/assets/gpu-performance-vs-power.png)

Also, there's a 16-core Neural Engine.

On security:

![Security]({{ site.url }}/assets/m1-security.png)

Universal App: x86_64 and arm64 binaries in a single `.app/`, falling back to use Rosetta 2 if there is no compatible binary.

MacBook Air, now fanless, incredible battery life, from $999.

![MacBook Air]({{ site.url }}/assets/macbook-air-overview.png)

Mac Mini, from $699.

13" MacBook Pro, from $1299:

![MacBook Pro]({{ site.url }}/assets/macbook-pro-overview.png)
